# 🌐 Network Traffic Analysis & Threat Detection Lab

## Overview
This project demonstrates hands-on network traffic analysis and threat detection using packet capture and analysis techniques in a controlled lab environment.

## Tools Used
- Wireshark
- TCP/IP analysis
- Linux

## Skills Demonstrated
- Packet inspection and analysis
- Network traffic monitoring
- Threat detection
- Protocol analysis

## Project Structure
- screenshots/ → Packet capture and analysis evidence
- pcaps/ → Raw network traffic files
- reports/ → Analysis report

## Investigation Highlights
- Identified suspicious IP communication patterns
- Analyzed abnormal traffic behavior
- Detected potential scanning and unauthorized access attempts

## Outcome
Successfully analyzed network traffic to identify and interpret potential security threats, demonstrating real-world network monitoring and detection skills.
